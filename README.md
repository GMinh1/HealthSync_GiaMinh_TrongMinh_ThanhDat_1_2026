# HealthSync App

## Giới thiệu

HealthSync là ứng dụng theo dõi sức khỏe được phát triển bằng Flutter. Ứng dụng giúp người dùng quản lý cân nặng, số bước chân và lượng calo tiêu thụ hàng ngày.

## Chức năng

* Quản lý dữ liệu sức khỏe
* Tính BMI
* Theo dõi lịch sử
* CRUD dữ liệu

## Công nghệ sử dụng

* Flutter
* Dart
